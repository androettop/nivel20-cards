# Nivel20 Cards

Crea **cartas imprimibles** para tus personajes de
[Nivel20](https://nivel20.com): conjuros, armas, armaduras, equipo y
dotes/rasgos, listas para recortar y jugar.

👉 **Úsalo aquí:** https://androettop.github.io/nivel20-cards/

## Cómo se usa

1. Abre tu personaje en Nivel20 y copia la dirección (URL) de la página.
2. Pégala en la app y pulsa **Generar cartas**.
3. Ajusta el tamaño, la separación y el redondeado de las cartas en el panel.
4. Pulsa **Imprimir** y elige escala 100% (sin «ajustar a página»).

Puedes pegar la URL del personaje tal cual, por ejemplo:

```
https://nivel20.com/games/dnd-2024/characters/1954502-kelsier
```

## Personaliza tus cartas

Desde el panel de configuración puedes cambiar:

- El **tamaño** de carta (póker, bridge, mini, tarot o a medida).
- La **separación** entre cartas y el **redondeado** de las esquinas.
- El grosor del borde, el margen de la hoja y el tamaño del texto.
- Qué **categorías** incluir (conjuros, armas, armaduras, equipo, dotes).
